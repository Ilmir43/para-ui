# para-ui

Мини-приложение на Vite + Solid для обзора проектов и ежедневников.

## Переменные окружения
Скопируй `.env.example` в `.env` и укажи пути, чтобы сразу видеть свои каталоги в интерфейсе:

```
VITE_PROJECTS_DIR="/Users/you/SecondBrain/Projects"
VITE_AREAS_DIR="/Users/you/SecondBrain/Areas"
VITE_DAILY_DIR="/Users/you/SecondBrain/Daily"
```

Приложение теперь пытается автоматически загрузить проекты и ежедневники, если по указанным путям доступен HTTP-каталог с файлом `index.json` вида:

```json
{ "files": ["project-a.md", "project-b.md"] }
```

Маршруты в `files` должны возвращать содержимое markdown-файлов. Если автоподгрузка не сработает, можно по-прежнему выбрать папки вручную через кнопки «Выбрать папку ...» в интерфейсе.
